# TypeScript Scripts Environment

## Setup

```bash
npm install
```

## Running Scripts

```bash
# Run a script once
npm start

# Run with watch mode (auto-reload on changes)
npm run dev

# Run any TypeScript file directly
npx tsx src/your-script.ts
```

## Type Checking

```bash
npm run typecheck
```

## Building to JavaScript

```bash
npm run build
```

Output will be in the `dist/` directory.
